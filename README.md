# SecureKey

## Developers: Ioanna Havryliuk, Poremska Liudmyla, Oleksandr Rozheliuk.

## Overview

SecureKey is a reliable command-line password manager that combines password strength checking, TOTP generation, and encrypted credential storage.

## Functionality

* Password strength checking (regex-based)
* TOTP (time-based one-time password) generation
* Encrypted credential vault protected by a master password
* Command-line operations to store, retrieve, list, and remove entries
* Secure handling of secrets and file permissions
